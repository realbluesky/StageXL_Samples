StageXL Samples

StageXL is a package for the Dart programming language. It offers an easy to use and complete API for impressive 2D content like games and other rich applications. The library started as an easy migration path for Adobe Flash developers to HTML5. Therefore the StageXL API is for the most part identical to the Flash API. Today many developers who have no prior experience with Flash are using StageXL for their applications.

Escape This game was developed with Adobe Flash and later ported to HTML5 with the help of Dart and StageXL. The whole process only took a few hours because no substantial code changes were necessary. The game uses bitmaps, sprites, sounds, text fields, event listeners, etc. try it
Supersonic Another game which was developed with Adobe Flash and later ported to HTML5. Just as with the game 'Escape' the source code was ported from ActionScript to Dart without much fuss. This game uses vector graphics, bitmaps, sounds, masks, etc. try it
Periodic Table This demo shows the periodic table of chemical elements. All display objects are drawn as vector graphics and cached as bitmaps for optimal performance. The animations are done with the Juggler animation framework which is fully integrated in StageXL. The stage is automatically scaled to fill the whole browser window. try it
Dart Logo This demo shows an animation of the Dart logo with a minimum amount of source code. The demo has 40 lines of code, but only half of it is necessary for the actual animated Dart logo. try it
Piano A virtual piano and four songs with a karaoke like experience. The demo shows the audio playback capabilities of StageXL in different browsers. Playing HTML5 audio can be tricky, but StageXL offers an easy to use cross browser Sound API (compatible to the Flash API). try it
Performance This demo can be used as a benchmark for StageXL. It is all about drawing small images and doing animations. As with all benchmarks, it's important to look at the use case and think about how those numbers apply to your use case. try it
FlipBook An example to show how to do sprite sheet animations. StageXL offers two essential building blocks - the FlipBook class to play sprite sheet animations and support for texture atlases. try it
Mask An example to show the support for arbitrary shaped masks StageXL. The example shows a field of flowers and you decide which mask should be applied. try it
RichTextField This demo uses the StageXL_RichTextField extension library to demonstrate multiple formats in a single canvas text field. See the extension readme for supported text-tags. try it
Flump Flump converts Flash keyframe animations into texture atlases and JSON that can be easily integrated into StageXL. The StageXL_Flump extension library loads those assets and plays the animations with minimum memory and cpu load. try it
